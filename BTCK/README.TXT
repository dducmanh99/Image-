1. Thực hiện thu thập bộ dữ liệu giống trong folder 'data'.
2. Sử dụng file 'lib.py' để tìm ma trận hiệu chỉnh ảnh. 
3. Sử dụng file 'main.py' để tìm tọa độ tâm vật theo phương pháp ppsd.png

Note: 
1. File 'calib_err.py' sử dụng để trực quan các sai số từ quá trình hiệu chỉnh. 
2. File 'process_data.py' để viết các kết quả vào txt để trực quan hóa. 

Phương pháp này chỉ sử dụng được cho bộ dữ liệu trong folder 'data' nên chỉ để tham khảo.
